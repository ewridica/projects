Файл Аналитика в Яндекс.Афише содержит описание и решение задачи оптимизации маркетинговых затрат.
Данные от Яндекс.Афиши с июня 2017 по конец мая 2018 года:
1. лог сервера с данными о посещениях сайта Яндекс.Афиши,
2. выгрузка всех заказов за этот период,
3. статистика рекламных расходов.

В файле произведен детальный анализ данных и даны дальнейшие рекомендации по оптимизации маркетинговых затрат.
